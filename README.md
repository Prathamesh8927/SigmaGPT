# SigmaGPT 🤖

SigmaGPT is a ChatGPT-inspired AI chatbot built using the MERN Stack.

## 🚀 Features

- AI Chat
- Multiple Chat Threads
- Chat History
- Gemini API
- MongoDB Storage

---

## 🛠 Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Gemini API

---

## 📦 Installation

### Backend

```bash
cd Backend
npm install
npm start
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
